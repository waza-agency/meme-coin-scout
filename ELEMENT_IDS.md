## Naming Convention

IDs follow the pattern: `[componente]-[función]-[numero]`

Examples:
- `header-nav-001` - Navigation element in header
- `sidebar-menu-002` - Menu item in sidebar
- `form-login-003` - Login form element
- `button-submit-004` - Submit button
